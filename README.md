# Soul Food

A social dining application powered by MEAN stack.

## Getting Started

### Prerequisities

```
node
mongo
```

### Installing

```
$ npm install
$ node server.js
```
## Built With

* NodeJS
* ExpressJS
* MongoDB
* AngularJS
* Socket.IO
* PassportJS
* Google Maps API
* Yelp API
